# Installation
```shell
 composer require mouhamedfd/sn-name-generator
 ```

# Code example
```php
use Mouhamedfd\Generator\SnNameGenerator as SnNmG;
```
## Generate fields
```php
$name=SnNmG::getName();
$firstName=SnNmG::getFirstName();
```
